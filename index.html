CurrencyIndex is a free JSON API for latest and historical currency, cryptocurrency and metal rates.

Update time for each element category is set according to schedule:

Category Update Time
Currency Every working day at 13:00 UTC
Local Currency Every hour on working days
Cryptocurrency Every 5 minutes
Metals Every 5 minutes on working days

USAGE

Each category has specific parameters and paths. In the next sections basic usage principles are desribed for each of them.

CURRENCY

Currency rates are loaded through the /baserates/ URI part.

The next section of the URI points defines time: latest/hitorical.

Example:

http://www.currencyindex.com/baserates/latest

This GET request will return all the latest rates of foreigh currencies.
